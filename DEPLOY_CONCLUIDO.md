## 📞 Suporte Rápido

### Erro "Firebase não reconhece o repositório"
- Verifique se o GitHub está autenticado
- Verifique o nome do repositório (case-sensitive)

### Erro "Publish directory não pode estar vazio"
- Use `.` (apenas um ponto) para publicar a raiz do projeto

### Deploy não aparece após 10 minutos
- Recarregue a página do Firebase Console
- Verifique em "Histórico de Deploy" se há erros
