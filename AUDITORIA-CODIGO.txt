## 📊 Estrutura de Dados Esperada

Cada atividade precisa ter:

```javascript
{
    // Campos tradicionais
    ticket: 1001,
    colaborador: "João Silva",
    dataInicio: "2026-05-06",
    horaInicio: "10:30",
    descricao: "Tarefa importante",
    
    // ✨ NOVOS CAMPOS DE AUDITORIA ✨
    usuario_id: "a1b2c3d4e5f6...",      // ID do Firebase Auth
    email_usuario: "joao@empresa.com",  // Email do usuário
    nome_usuario: "João Silva",         // Nome completo
    data_criacao: "2026-05-06T10:30:45.123Z",  // ISO 8601
    data_atualizacao: "2026-05-06T14:30:00.456Z"  // ISO 8601 (opcional)
}
```

## 🔍 Filtros Implementados

### Filtro 1: Usuário
```javascript
if (usuarioSelecionado) {
    dados = dados.filter(a => a.usuario_id === usuarioSelecionado);
}
```

### Filtro 2: Data Inicial
```javascript
if (dataInicio) {
    dados = dados.filter(a => {
        const dataAtiv = a.data_criacao 
            ? new Date(a.data_criacao).toISOString().split('T')[0]
            : a.dataInicio;
        return dataAtiv >= dataInicio;
    });
}
```

### Filtro 3: Data Final
```javascript
if (dataFim) {
    dados = dados.filter(a => {
        const dataAtiv = a.data_criacao 
            ? new Date(a.data_criacao).toISOString().split('T')[0]
            : a.dataInicio;
        return dataAtiv <= dataFim;
    });
}
```

## 🐛 Troubleshooting

### Problema 1: Dropdown vazio
- **Causa**: Sem dados em Firebase → usuarios/
- **Solução**: Crie pelo menos uma conta

### Problema 2: Tabela vazia
- **Causa**: Sem atividades ou sem campos de auditoria
- **Solução**: Crie atividades em index.html após ativar auditoria

### Problema 3: CSV não baixa
- **Causa**: Navegador bloqueando downloads
- **Solução**: Verifique permissões ou tente em incógnito

### Problema 4: Datas estranhas
- **Causa**: Timezone diferente
- **Solução**: Esperado. Converte para horário local automaticamente
