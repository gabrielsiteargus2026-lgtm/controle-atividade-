API - Ranking de Colaboradores

Endpoint: GET /api/ranking?data=DD/MM/YYYY

Descrição: Retorna o ranking de colaboradores com mais atividades no dia especificado

Atualização: Em tempo real (a cada 2 segundos)

Exemplos de Uso:

JavaScript (Fetch):
fetch('/api/ranking?data=24/04/2026')
  .then(r => r.json())
  .then(data => console.log(data))
